◆この製品の音源は下記のイメージで制作しております。

▼game_in
・break_and_chain : ブロックを壊す音、連鎖して消える時の音です。
・move            : ブロックの移動、回転する時の音です。
・acceleration    : ブロックが落ちる速度を速める時の音です。
・drop            : ブロックが下のブロックに到達したときの音です。
・immovable       : ブロックが障害物に邪魔されて移動や回転が出来なかった時の音です。

▼system
・cursor          : タイトル画面やメニュー画面のカーソルを動かしたときの音です。
・select          : メニュー画面などでボタンを押した時の音です。
・cancel          : キャンセルした時や前の画面に戻る時の音です。
・pause           : ゲームプレイ中にセレクトボタンを押した音です。
・score           : ブロックを破壊した時に得点が加算されるときの音です。
・countdown       : ゲームの開始時に鳴る音です。
・game_clear      : ゲームを攻略した時の音です。
・game_over       : プレイヤーが負けた時の音です。
